# Buttons Styling of buttons and anchors. - `btn` styles layout, dimensions of button - content of button must be placed inside `<span>` - icon for button must be placed in `<span>` with either css classes `fa`, `fas`, `fab`, `far`, `glyphicon` - disabled button can be displayed using `disabled` css class or attribute (for anchor you can use only css class) - `btn-*` adds coloring to button, according meaning, must be used together with `btn` class - `primary` - `secondary` - `info` - `default` - `success` - `warning` - `danger` - `error` - `btn-icon` - special class that can be used with or without `btn` class - `buttons-gap` - gaps between buttons, requires *flexbox* ## `btn` css class - **properties** - *letter spacing* - *background image* set to `none` - *text shadow* - *box shadow* - *text transformation* - *transition* - *border* - *border radius* - *padding* - *font-family* - *cursor* set to `pointer` - *white space* set to `nowrap` - *font size* - *font weight* - *min width* - *position* set to `relative` - *display* set to `flex` - *flex direction* set to `row` - *gap* - **customization** - *letter spacing* => `button.letterSpacing` - *text shadow* => `button.textShadow` - *box shadow* => `button.boxShadow` - *text transformation* => `button.textTransformation` - *transition* => `button.transition` - *border* => `button.border` - *border radius* => `button.borderRadius` - *padding* => `button.padding` - *font size* => `button.fontSize` - *font-family* => `fontFamily` - *font weight* => `button.fontWeight` - *min width* => `button.minWidth` - *gap* => `button.contentGap` - *scrollbar color* => `primary.scrollbar` ### **EXAMPLE `btn` only**
## `btn-*` css class - **sub selectors** - *::before* allows styling of background - **properties** - *content* set to `""` - *position* set to `absolute` - *transition* ### **EXAMPLE `btn-primary`**
