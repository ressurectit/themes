# Blocks Visual building blocks for creating content. - `primary-block` primary block used for displaying main content - `secondary-block` secondary block used for displaying main content with in different look - `highlight-block` allows highlighting content inside of primary or secondary block ## Primary block - **properties** - *padding* top and bottom - *background color* - *font color* - *scrollbar color* - **customization** - *background color* => `primary.background` - *font color* => `primary.foreground` - *scrollbar color* => `primary.scrollbar` - **global** - *padding* => `sizeXs`
